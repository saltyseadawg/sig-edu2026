# Checklist

https://acl-org.github.io/ACLPUB/formatting.html#appendices


## Paper Length
- [x]  Paper length is <= 8 pages
    - excludes References, Appendices, Ethics, limitations 

## File Format
- [x] Paper is in PDF format
- [x] PDF file embeds all necessary fonts
  - check with pdffonts mypaper.pdf (want all "yes" under "emb")
- [x] Paper uses A4 paper format
  - pdfinfo mypaper.pdf (should be 595.276 x 841.89 pts)

## Paper Format

### Title and Authors
- [x] Title is in title case.

### Layout
- [x] All text (except pg numbers) must fit within margins
- [x] Pages are numbered **in review version**
### Abstract
- [x] Abstract is max 200 words

### Text
- [ ] Indent 0.4 cm when starting a new paragraph, except for first paragraph in a section

### Figures and Tables
- [x] Provide a caption for every figure/table

### Limitations
- [x] Paper includes Limitations section
- [x] Limitations placed after Conclusion and before references

### Ethical Considerations
- [x] Should be placed after conclusion and before references

### Acknowledgements
- [x] **Do not include this section in review version**

## Appendices
- [x] Letter appendices and provide informative title


## General APA Styling Guidelines
- [ ] Punctuation goes outside of parantheses when ending a clause
- [ ] Citations of several authors separated by semi-colons
